동작 방식(PreLotto_re.py)

- 같을 경우: 중복 방지를 위해 추가하지 않음
- 이후 Tensorflow안에 내재된 Keras 모델을 통해 직전 회차 데이터를 기반으로 다음 회차의 번호를 예측합니다

   - create_dataset() 함수를 통해 과거 회차 데이터를 시계열 입력 형태로 변환합니다
